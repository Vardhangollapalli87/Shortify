# Shortify

A production-grade URL Shortener SaaS platform built with Node.js, Express, MongoDB Atlas, Redis Cloud, React, and Google OAuth.

## Features

### Authentication

* Email & Password Authentication
* JWT Access Tokens
* Refresh Token Rotation
* Google OAuth Login
* Password Reset Flow
* Email Verification
* Resend Transactional Emails

### URL Management

* Create Short URLs
* Custom Aliases
* Expiring Links
* Password Protected Links
* URL Ownership

### Analytics

* Click Tracking
* Unique Visitors
* Browser Analytics
* Device Analytics
* Operating System Analytics
* Referrer Analytics
* Time-Series Analytics

### Product Experience

* Professional responsive SaaS workspace
* Light and dark themes with saved preferences
* Accessible keyboard focus and navigation states
* Mobile navigation drawer and responsive link cards
* Dashboard, analytics, links, QR, and account settings experiences

### Infrastructure

* MongoDB Atlas
* Redis Cloud
* Docker
* GitHub Actions
* Render Deployment
* Vercel Deployment
