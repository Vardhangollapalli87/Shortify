# Shortify

A production-grade URL Shortener SaaS platform built with Node.js, Express, MongoDB Atlas, Redis Cloud, React, and Google OAuth.

## Features

### Authentication

* Email & Password Authentication
* JWT Access Tokens
* Refresh Token Rotation
* Google OAuth Login
* Password Reset Flow

### URL Management

* Create Short URLs
* Custom Aliases
* Expiring Links
* Password Protected Links
* URL Ownership

### Analytics

* Click Tracking
* Unique Visitors
* Browser Analytics
* Device Analytics
* Operating System Analytics
* Referrer Analytics
* Time-Series Analytics

### Infrastructure

* MongoDB Atlas
* Redis Cloud
* Docker
* GitHub Actions
* Render Deployment
* Vercel Deployment
